# 📌 Project Overview

Healthcare organizations generate massive volumes of patient and operational data every day.  
Without proper analytics, it becomes difficult to monitor:

- Patient admission trends
- Hospital operational performance
- Medical condition distribution
- Insurance provider billing
- Revenue & cost patterns
- Healthcare service efficiency

This project converts raw healthcare datasets into meaningful business insights through **SQL analysis, KPI reporting, and interactive Power BI dashboards**.
